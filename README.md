# Habi

Habi is a social learning platform designed to make skill-building feel social, motivating, and interactive.

Users can learn new skills through personalized video feeds, upload progress updates, build streaks, unlock achievements, and connect with friends learning similar skills.

---

## Features

### Personalized Learning Feed
- Skill-based onboarding flow
- Dynamic YouTube API integration
- Personalized skill video recommendations
- Search functionality for tutorials and content
- Multiple skill support

### Social Features
- User profiles
- Profile photo system
- Likes and saved videos
- Friend search system
- Shared progress uploads
- Settings modal
- Username customization

### Progress Tracking
- Daily streak tracking
- Achievement badge system
- Progress photo/video uploads
- Skill-based learning organization
- Gamified learning experience

### Authentication & User System
- Firebase Authentication
- Username system
- Username cooldown protection
- Firebase Firestore database
- Firebase Storage uploads

---

## Tech Stack

- React
- Vite
- Firebase Authentication
- Firestore
- Firebase Storage
- YouTube Data API
- CSS

Fully client-side application — Firebase handles authentication, database, and storage functionality.
